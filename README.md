# FRONT_ServerDeVoeux
Front Serveur de voeux projet RD

## Installation du projet
```
yarn install
```
ou
```
npm install
```
### Compiler et lancer le serveur du projet en local
```
yarn serve
```
ou
```
vue-cli-service serve
```
### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
